# react-native-hr

react-native-hr brings htmls' `<hr />` to react-native. <br />
It also brings some extra functionality:
  <ul>
    <li>Overayling text onto the line.</li>
    <li>Changing text colour.</li>
    <li>Changing line colour.</li>
  </ul>
